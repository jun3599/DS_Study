# 네이버 Open API를 활용한 Daily News DashBord 
매일의 뉴스를 자연어처리 방법에 따라 요약 정리하고, 주요 정보를 사용자에게 알려주는 서비스를 제작합니다.   
해당 서비스는 이미 여러 곳에서 제공 중이지만, 자연어 처리를 공부하는 가장 재미있는 방법이 될 것입니다.   

## Naver Developers Open Api 
* 해당 프로그램은 [Naver Developers | 검색](https://developers.naver.com/docs/common/openapiguide/apilist.md#%EB%A1%9C%EA%B7%B8%EC%9D%B8-%EB%B0%A9%EC%8B%9D-%EC%98%A4%ED%94%88-api) 에서 제공하는 공공 API를 활용해 제작되었습니다.
